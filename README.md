# RescueRelief AI (Demo)

This is a **frontend-only** demo for a National Disaster Management Dashboard.
All data is saved in your browser using **LocalStorage** (no backend).

## Run

- Open `index.html` in any browser (Chrome/Edge recommended).

## Features (clickable & usable)

- **Navigation**: Home / Disasters / Resources / Alerts / Contact
- **Disasters**: add, edit, resolve/reopen, search/filter
- **Resources**: inventory adjust (+/−), quick replenish, allocate to a disaster (creates log entry)
- **Alerts**: create, preview broadcast, acknowledge/unacknowledge, delete, filter
- **Contact**: submit message and view saved messages

## Reset

Use the **Reset** button in the top-right to restore default demo data.
